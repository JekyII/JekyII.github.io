Case Study

Portfolio Website Refresh

Elevating my personal cybersecurity portfolio with immersive visuals, dynamic interactions, and a cohesive blog system.

Overview

The portfolio refresh project modernises my personal site with a cyberpunk-inspired aesthetic, animated matrix hover effects, and a revitalised content strategy. The work spans design, front-end engineering, and automation around blog content to keep the site current and engaging.

Role

Solo designer & developer

Timeline

Aug – Oct 2025

Tools

VS Code, Figma, Tailwind-inspired utility CSS, vanilla JS

Focus Areas

Experience design, performance, accessibility, automation

Goals

  • • Craft a memorable first impression that aligns with security-focused storytelling.
  • • Streamline header, footer, and contact UX for easier navigation and outreach.
  • • Establish maintainable systems for blog posts, project pages, and animations.

Implementation

I rebuilt the site with modular HTML partials for the header, footer, and contact form, then connected them through lightweight JavaScript loaders. The visual system leans on glassmorphism cards, gradient accents, and a custom matrix proximity effect that animates headings, navigation tabs, and body copy without impacting readability.

The blog leverages a JSON manifest to dynamically render cards and individual posts, while responsive CSS keeps typography and spacing consistent across breakpoints. Accessibility was considered throughout, from keyboard-friendly focus states to respecting reduced-motion preferences.

Behind the scenes, I prepared automation hooks for future scripted blog image generation, and ensured integrations like reCAPTCHA can be themed and swapped easily once production keys are configured.

Outcomes

  • Cohesive visual identity anchored by neon-accented cyber motifs.
  • Faster iteration by extracting reusable components and utilities.
  • Improved engagement through interactive blog listings and project previews.
  • Clear roadmap for future automation (blog images, contact workflows).

Next Steps

Remaining work includes plugging in the production reCAPTCHA key, extending the blog asset generator, and expanding the project catalogue with deeper technical breakdowns and metrics.
